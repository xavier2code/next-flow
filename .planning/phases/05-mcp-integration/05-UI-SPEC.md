## Not Applicable

**Phase 5 (MCP Integration) is a backend-only phase with no frontend deliverables.**

This phase implements:
- MCP Client with Streamable HTTP transport and SSE fallback (MCP-01)
- MCP Manager for server lifecycle, health monitoring, reconnection (MCP-02)
- MCP tool discovery via tools/list and registration in Tool Registry (MCP-03)
- Admin API CRUD endpoints for MCP server management (MCP-04)
- MCP tool invocation through Tool Registry routing (MCP-05)

All UI requirements for MCP management are deferred to Phase 7 (Frontend), specifically:
- **UI-10**: MCP management UI (server registration, connection status, tool list)

## API Response Format Contract

While not visual, the Admin API (MCP-04) establishes response contracts that Phase 7 UI will consume:

| Endpoint | Method | Response Shape |
|----------|--------|----------------|
| /mcp-servers | POST | `EnvelopeResponse[MCPServerResponse]` -- status 201, status="connecting" |
| /mcp-servers | GET | `PaginatedResponse[MCPServerResponse]` -- cursor pagination |
| /mcp-servers/{id} | GET | `EnvelopeResponse[MCPServerResponse]` |
| /mcp-servers/{id} | PATCH | `EnvelopeResponse[MCPServerResponse]` -- triggers reconnect |
| /mcp-servers/{id} | DELETE | status 204 -- deregisters tools, disconnects client |
| /mcp-servers/{id}/tools | GET | `EnvelopeResponse[list[MCPToolResponse]]` |

### MCPServerResponse Fields (for Phase 7 reference)

| Field | Type | Values |
|-------|------|--------|
| id | UUID | Unique server identifier |
| name | string | Human-readable server name |
| url | string | Server endpoint URL |
| transport_type | string | "streamable_http" or "sse" |
| status | string | "connecting", "connected", "disconnected", "error" |
| config | object | Server-specific configuration |
| created_at | datetime | Registration timestamp |
| updated_at | datetime | Last update timestamp |

### MCPToolResponse Fields (for Phase 7 reference)

| Field | Type | Purpose |
|-------|------|---------|
| name | string | Tool name (without namespace prefix) |
| namespaced_name | string | Full name: mcp__{server}__{tool} |
| description | string | Tool description from MCP server |
| input_schema | object | JSON Schema from tools/list |
