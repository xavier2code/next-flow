    Create the FastAPI application with lifespan management, database session factory, Redis client, dependency injection, and health check.

    **1. Create `backend/app/db/session.py`** per D-05, D-06:
    - Import `create_async_engine`, `async_sessionmaker`, `AsyncSession` from `sqlalchemy.ext.asyncio`
    - Import `settings` from `app.core.config`
    - Create `engine = create_async_engine(settings.database_url, echo=settings.debug, pool_size=20, max_overflow=10)`
    - Create `async_session_factory = async_sessionmaker(engine, class_=AsyncSession, expire_on_commit=False)`
    - Create `get_db()` async generator dependency:
      ```python
      async def get_db() -> AsyncGenerator[AsyncSession, None]:
          async with async_session_factory() as session:
              try:
                  yield session
                  await session.commit()
              except Exception:
                  await session.rollback()
                  raise
      ```

    **2. Create `backend/app/db/redis.py`** per D-10, D-11, D-12:
    - Import `redis.asyncio as aioredis`
    - Import `settings` from `app.core.config`
    - Create `get_redis()` function that returns the Redis instance from `app.state.redis` (set during lifespan)
    - Key naming will follow convention `nextflow:{domain}:{key}` (documented as a constant `KEY_PREFIX = "nextflow"`)

    **3. Create `backend/app/api/deps.py`:**
    - Re-export `get_db` from `app.db.session`
    - Re-export `get_redis` from `app.db.redis`
    - This file is the single import point for all route dependencies

    **4. Create `backend/app/api/v1/health.py`:**

      - Return `{"status": "healthy", "redis": "connected"}` on success

      - Shutdown: close Redis (`await app.state.redis.close()`), dispose engine (`await engine.dispose()`)

    - Include v1 router: `app.include_router(v1_router)`
